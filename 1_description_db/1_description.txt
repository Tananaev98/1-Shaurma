3) reserves Таблица которая хранит в себе шаблоны объектов "Запасы" на основаниии этих объектов создаются поступления товаров. 

	Структура:
	
	id формат: int; назначение: первичный ключ; параметры: уникальное значение, авто увеличение
	
	name формат: varchar50; назначение: наименование объекта меню; параметры: не может быть пустым 	
	
	min_quantity формат: int; назначение: минимальное количество в котором товар может поступать; параметры: не может быть пустым
	
	max_quantity формат: int; назначение: максимальное количество в котором товар может поступать; параметры: не может быть пустым
	
	critical_quantity формат: int; назначение: Критическое количество товара при котором директор должен быть уведомлен; параметры: не может быть пустым

    availability формат: varchar50; назначение: доступен ли объект для использования(создания на основании его поступлений); параметры: по умолчанию должен быть в значении "Не доступен" 	




4) remains_reserves Таблица которая хранит в себе "Поступления товаров" который создаются на основании данных из таблицы reserves

	Структура:
	
	id формат: int; назначение: первичный ключ; параметры: уникальное значение, авто увеличение
	
	id_reserves формат: int; назначение: внешний ключ; параметры: не может быть пустым
	
	date_of_receipt формат: date; назначение: дата поступления товара; параметры: не может быть пустым 	
	
	cost_gram формат: int; назначение: стоимость товара за один грам; параметры: не может быть пустым
	
	quant_received формат: int; назначение: количество в котором товар поступил; параметры: не может быть пустым
	
	quant_spent формат: int; назначение: количество товара в котором он израсходован; параметры: не может быть пустым

	expiration_date формат: date; назначение: дата истечения срока годности; параметры: не может быть пустым 	

    status формат: varchar50; назначение: Пригоден для использования, просрочен, израсходован, срок истекает, не доступен; параметры: по умолчанию должен быть в значении "Не доступен"

	quant_reserv формат: int; назначение: Количество; параметры: не может быть пустым
